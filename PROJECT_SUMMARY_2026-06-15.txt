🔧 CONFIGURATION
================================================================================

Main Config File: config/monitoring.conf

Key Settings:
  LOG_PATHS              - Files to monitor (array)
  ERROR_PATTERN          - Regex for errors
  WARN_PATTERN           - Regex for warnings
  ALERT_THRESHOLD_ERROR  - Trigger threshold
  ALERT_TIME_WINDOW      - Detection window (seconds)
  WEBHOOK_URL            - Slack webhook
  RETENTION_DAYS         - Log retention period
  CHECK_INTERVAL         - Monitoring interval
  ENABLE_WEBHOOK         - Toggle notifications
  DEBUG_MODE             - Debug output

Example Configuration:
  LOG_PATHS=("/var/log/syslog" "/var/log/app.log")
  ERROR_PATTERN="ERROR|CRITICAL|FATAL|Exception"
  ALERT_THRESHOLD_ERROR=10
  ALERT_TIME_WINDOW=300
  WEBHOOK_URL="https://hooks.slack.com/services/..."
