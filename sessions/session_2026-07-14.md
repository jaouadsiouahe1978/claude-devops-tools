## 🔑 Key Concepts Enseignés

### Handlers (Innovation Clé)
```yaml
- notify: restart nginx  # Triggered par tâche
handlers:
  - name: restart nginx  # Exécute SEULEMENT si changement
```

### Variables Hierarchy
```
Command-line (-e) > Play vars > Registered > 
Host vars > Group vars > Role vars > Defaults
```

### Jinja2 Templates
```jinja2
listen {{ item.listen | default('80') }};
worker_processes {{ nginx_worker_processes }};
{% if ssl_enabled %}...{% endif %}
```

### Idempotence
- Tasks are **safe to run multiple times**
- No side effects if nothing changed
- Foundation of reliable automation

### Tags
```bash
ansible-playbook playbook.yml --tags webserver
ansible-playbook playbook.yml --tags database
```

### Niveau 1: Docker Testing

### Niveau 2: Add Custom Site
